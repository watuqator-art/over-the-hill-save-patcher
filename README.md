# Over the Hill (Demo) — Save Patcher

Графический редактор сохранения для демо **Over the Hill**. Выбираешь папку сохранений,
отмечаешь нужные патчи галочками, жмёшь «Применить». Перед каждым применением делается
авто-бэкап, есть кнопка отката.

## Для пользователя

1. Скачать **`OverTheHillPatcher.exe`** (папка `dist/`).
2. **Закрыть игру полностью** (иначе автосейв затрёт изменения).
3. Запустить exe. Установка не нужна — это автономный файл.
4. Папка сохранений (`…/Documents/My Games/over the hill`) и профиль определяются
   автоматически; при необходимости укажи вручную кнопкой «Обзор…».
5. Отметить патчи → «Создать бэкап и применить».
6. Запустить игру и проверить.

### Патчи
- Деньги → 9 999 999
- Вся косметика / аксессуары (134)
- Все покраски / ливреи (199)
- Все инструменты (×99)
- Ездовые машины demo (3: Defender, FJ40_Truck, Blazer)
- Открыть все хижины (6) — стартовые точки
- Открыть все точки интереса (18)
- Открыть смотровые точки (2)
- Снять весь туман войны (открыть карту)

### Важно
- **Steam Cloud** может откатить правки при запуске. Если прогресс вернулся — отключи
  облако для демки в свойствах игры в Steam и примени заново.
- Поддерживается только **демо-сборка**. ID предметов зашиты в `data.json`; в другой
  версии игры они могут отличаться.
- Кнопка «Восстановить из бэкапа» возвращает последний бэкап, сделанный патчером.

## Для разработчика (пересборка)

Требуется Python 3 + зависимости:

```
pip install pyinstaller psutil customtkinter
build.bat
```

Готовый exe появится в `dist/OverTheHillPatcher.exe`. Данные предметов лежат в `data.json`
и вшиваются в exe при сборке.
